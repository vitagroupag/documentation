###############
Getting Started
###############

.. warning:: WIP

This chapter aims to give an overview an "hello world" example for software developers
to build applications based on EHRbase and the openEHR specification. While we use
EHRbase as the backend, the example will actually run against every other conformant
openEHR implementation that implements the official openEHR REST API.
